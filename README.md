## Install

1. Clone the repository:
```sh
git clone https://github.com/iagoestevezvigo/speaking_globe_opensource.git
```
2. Change to the repository folder
```sh
cd speaking_globe_opensource
```
3. Get dependencies
```sh
flutter pub get
```
4. Run the app
```sh
flutter run
```
